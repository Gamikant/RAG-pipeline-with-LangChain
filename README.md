## RAG (Retrieval Augmented Generation) using LangChain
In this this project, I've locally implemented RAG using Langchain. 

**Requirements**:
1. Python 3.13+
2. Groq account - for free LLM API key

**First create a Python Virtual Environment:**
1. Download the `Python` extension from the VS Code extension store.
2. `Ctrl + Shift + P` - to open top menu
3. Search for `Python: Create Environment...` and create a `.venv` virtual environment using Python 3.13+
4. Choose `requirements.txt` before creating the venv for the packages to get automatically installed. But if you missed it, run this command in your project root cmd prompt - `pip install -r requirements.txt`

**Running the Chat Application**:
```bash
## Run this command in the cmd prompt inside project root
python app.py
```
